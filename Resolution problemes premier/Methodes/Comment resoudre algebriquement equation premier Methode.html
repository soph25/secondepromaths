Étapes

1. Appliquer les mêmes additions ou soustractions dans les deux membres de l'équation pour isoler l'inconnue dans l'un des membres.

2. Appliquer la même multiplication ou division dans les deux membres de l'équation pour obtenir l'inconnue.

Exemple
`3x+7= -x+3`

  • On ajoute  `x`  dans chaque membre pour éliminer le terme en  \(x\)  du membre de droite :

\(3x+7+x=-x+3+x\)

`4x+7=3`

  • On soustrait 7 dans chaque membre afin de n'avoir que le terme en  \(x\) dans le membre de gauche : 

\(4x+7-7=3-7\)

\(4x=-4\)

  • On divise par 4 chaque membre afin d'éliminer le coefficient (4) devant \(x\)  : 

\(\dfrac{4x}{4}=\dfrac{-4}{4}\)

\(x=-1\)
