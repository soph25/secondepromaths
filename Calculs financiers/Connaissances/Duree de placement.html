Définition

La durée de placement est le temps pendant lequel un capital initial est placé à un certain taux avant d'être retiré. Il est calculé avec la formule :
  `n=i/(C \times t`

  • `n`  : durée de placement (en année) ;
  • `i`   : intérêt simple (en €) ;
  • `C`  : capital initial (en €) ;
  • `t`  : taux annuel de placement.
