Exemple

Pour un intérêt de 125 €, une durée de 2 ans et un capital de 400 €, le taux annuel de placement est :

    \(t = \dfrac{125}{400×2} = 0{,}0156\)

En %, \(t = 15{,}6~\%\) .

Exercice

Calculer le taux annuel de placement `t`  pour les valeurs suivantes.

1.  \(C = 510~€~; n = 4~\text{ans}~; i = 120~€\) .

2. \(C = 432~€~; n = 6~\text{ans}~; i = 315~€\) .

3.  \(C = 8~000~€~; n = 6~\text{ans}~; i = 2~250~€\) .

4. \(C = 1~575~€~; n = 15~\text{ans}~; i = 3~245~€\) .

5. \(C = 2~589~€~; n = 7~\text{ans}~; i = 1~200~€\) .
