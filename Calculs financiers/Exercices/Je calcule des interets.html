Exemple

Un capital de 1 000 € placé à un taux de 5 % pendant 3 ans. 

  \(i = 1~000 \times 0,05 \times 3 = 150~ €\)

Exercice

Calculez les intérêts  `i` pour : 

a. \(C = 500~€~; t = 10~\%~; n = 7~\text{ans}\)  
`` b. \(C = 250~€~; t = 15~\%~; n = 2~\text{ans}\)

c. \(C = 600~€~; t = 8~\%~; n = 4~\text{ans}\)

d. \(C = 5~000~€~; t = 11~\%~; n = 12~\text{ans}\)

e. \(C = 3~500~€~; t = 6~\%~; n = 8~\text{ans}\)
