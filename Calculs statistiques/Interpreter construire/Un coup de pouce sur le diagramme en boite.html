Le diagramme en boîte, également appelé diagramme en boîte à moustaches, illustre différents indicateurs d'une série statistique :

  • la valeur minimale ( \(Min\) ) ;
  • la valeur maximale ( \(Max\) ) ;
  • la médiane ( `M_e` ) ;
  • le premier quartile ( `Q_1` ) ;
  • le troisième quartile ( `Q_3` ).
