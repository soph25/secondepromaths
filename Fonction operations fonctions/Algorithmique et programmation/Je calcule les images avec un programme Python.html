Écrire un programme Python dans lequel : 

  • on demande à l'utilisateur la valeur de  `x`  ;
  • si  `x<0`  la fonction  `f(x)` est telle que  `f(x)=x²`  ;
  • si  `x>0`  la fonction  `f(x)`  est telle que  `f(x)=2x`  ;
  • la valeur de l'image est ensuite affichée.
