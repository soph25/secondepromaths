Compétences mobilisées

  • S'approprier : comprendre l'énoncé et relever les informations importantes.
  • Analyser : proposer une méthode de résolution.
  • Réaliser : résoudre par une méthode algébrique ou graphique une équation du type  \(f(x)=c\)  ou une inéquation du type \(f(x) où \(c\)  est un réel donné et  \(f\)  une fonction affine ou une fonction du type \(x ↦ kx²\) (avec  \(k\) réel donné).
  • Valider : faire le lien avec la ou les valeur(s) de l'équation et la problématique posée ; vérifier leur cohérence.
  • Communiquer : rédiger une phrase de conclusion ; respecter les écritures mathématiques ; utiliser les termes mathématiques ; expliciter le raisonnement.
