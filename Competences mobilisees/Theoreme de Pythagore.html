Compétences mobilisées

  • S'approprier : reconnaître les codages ; identifier l'hypoténuse ; reconnaître les différents types de triangle.
  • Analyser : proposer une méthode permettant de prouver qu'un triangle est rectangle ; calculer une troisième longueur.
  •  Réaliser : résoudre des équations de type \(ax+b\)  ; calculer le carré de valeurs ; résoudre des équations de type \(x^2=a\) .
  • Valider : vérifier la véracité d'une égalité.
  • Communiquer : rédiger une phrase de conclusion ; respecter les écritures mathématiques ; utiliser les termes mathématiques ; expliciter le raisonnement.
