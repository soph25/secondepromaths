Comment utiliser le format pourcentage dans Excel - Méthode

1. Sélectionnez les cellules.


2. Dans le sous-menu « Nombre » du menu « Accueil », sélectionnez le format pourcentage.

3. Saisissez les nombres sans le signe %. Celui-ci est ajouté automatiquement.
