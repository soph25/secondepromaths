Rédiger un programme Python dans lequel : 

  • l'utilisateur donne les coordonnées d'un point  `\text{A}(x_\text{A}; y_\text{A})`  et d'un point  \(\text{B}(x_\text{B}; y_\text{B})\)  ;
  • le calcul des coefficients  `a`  et  `b`  est effectué dans le programme à partir des coordonnées données ;
  • les valeurs de  `a`  et de  `b` sont affichées.
