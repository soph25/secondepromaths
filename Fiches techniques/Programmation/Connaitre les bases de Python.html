 4. Opérations arithmétiques

5. Les instructions conditionnelles permettent d'exécuter différentes parties du code en fonction de conditions spécifiques.

  • L'instruction if introduit une condition. Si la condition est est vérifiée, le code indenté (en retrait) sous if est exécuté.
  • L'instruction elif est une abréviation de « else if ». Elle introduit une condition supplémentaire si la condition précédente n'est pas vraie.
  • L'instruction else introduit une partie de code qui est exécutée si aucune des conditions précédentes n'est vraie.

7. Les opérateurs de membres

8. Les opérateurs logiques sont utilisés pour combiner des conditions logiques.

  • not : inverse la valeur logique d'une condition.
  • and : renvoie « Vrai » si toutes les conditions sont vraies.
  • or : renvoie « Vrai » si au moins une des conditions est vraie.
