1. Entrer les données.

Lorsque vous arrivez dans l’application « Régressions », vous devez entrer vos données dans un tableau à deux colonnes. Vous pouvez ajouter jusqu’à trois tableaux de données.



Dans la première colonne (X1), renseigner les valeurs de la première variable de votre série.

Dans la seconde colonne (Y1), renseigner les valeurs de la seconde variable de votre série.

2. Afficher le nuage de points correspondant aux données.

Une fois que vous avez entré vos données dans le tableau de l’onglet « Données », vous pouvez afficher le nuage de points correspondant.

  • Sélectionner l’onglet « Graphique »  en haut de l’écran en vous déplaçant avec les flèches.

  • Valider en appuyant sur la touche EXE.

Vous visualisez alors les points qui représentent vos données.

3. Afficher un modèle de régression ou de modélisation.

Une fois le nuage de points affiché, vous pouvez ajouter un modèle de régression.

  • Lorsque vous êtes dans l’onglet «  Graphique », appuyer sur OK  pour afficher la liste des modèles de régression disponibles.
  • Sélectionner ensuite le modèle que vous souhaitez utiliser : LINEAIRE et appuyez sur OK  pour valider.
  • Aller dans « REGRESSIONS » puis cliquer sur entrée : l’équation de régression s’affiche.
