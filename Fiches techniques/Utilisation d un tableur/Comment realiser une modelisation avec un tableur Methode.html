1. Entrer les données

  • Entrer les valeur de x dans la colonne A.
  • Entrer les valeurs de y dans la colonne B.

2. Afficher le nuage de points correspondant aux données

  • Sélectionner avec la souris  la plage contenant toutes les données (les cellules se grisent).

  • Aller dans « Insertion » puis sélectionner « Nuage de points », sans relier les points  pour représenter la série.

3. Afficher un modèle de régression ou de modélisation

  • Faire un clic droit sur un des points du nuage (double-cliquez pour avoir un cadre gris).

  •  Cliquer sur « Ajouter une courbe de tendance ».
  •  Sélectionner « Linéaire ».

  • Sélectionner « Afficher l’équation sur le graphique » puis valider : l'équation s'affiche sur le graphique.
